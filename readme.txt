La aplicación cuenta con un directorio para la parte frontend y otra para la parte backend.
	
	1. La parte frontend se encuentra en la carpeta iaw2017-frontend. Se implementó con AngularJS y se debe correr el comando: npm install -g grunt-cli
	Luego para levantar el servidor se debe correr el comando grunt serve.
	2. La parte backend se encuentra en la carpeta iaw2017-backend. Se implementó con NodeJS usando MongoDB como base de datos. Para correrlo debe correrse primero el comando npm install y levantarse Mongo. Luego, se puede levantar el servidor con el comando npm start.
